*本方法采用hexo-tag-aplayer,未提及部分可自行前往作者处查阅。

安装

bash

使用aplayer

不出意外安装完毕后，就可以进行使用了。
(如果你想添加自己的网易云或者其他家的已有歌单，建议直接阅读第三部分，简单。)
本文以 /source/music/index.md 修改为例，看如何将歌曲及列表插入到我们的网页中。

-url: 音乐文件 URL 地址

-narrow: （可选）播放器袖珍风格

-width:xxx: (可选) 播放器宽度 (默认: 100%)
-lrc:xxx: （可选）歌词文件 URL 地址

Code
